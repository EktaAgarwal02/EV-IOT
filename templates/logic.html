Project Logic

1. Data Collection

IoT-enabled EVs send sensor data every 15 minutes, including battery voltage, motor torque, RPM, temperature, and vibration signals.

2. Data Preprocessing

The raw data is cleaned, normalized, and missing values are handled. Features like RUL (Remaining Useful Life) and TTF (Time to Failure) are derived.

3. Model Training

Machine learning models are trained on historical EV datasets to predict failure probability, maintenance needs, and battery health status.

4. Prediction & Monitoring

Real-time data is fed into the trained model, which provides early failure warnings and maintenance recommendations for safe EV operation.
